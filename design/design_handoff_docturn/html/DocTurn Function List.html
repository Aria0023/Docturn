Function List · functional decomposition

Functional decomposition — what the system does, broken into reviewable units.

Seventeen top-level functional areas (F1–F17) decomposed into ~96 leaf functions. Each leaf names the subsystem that owns it and whether it ships in the built v1 core or arrives with the target expansion (milestone). This is the bridge between the use-case model and the system requirements.
